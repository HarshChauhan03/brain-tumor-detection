# 🧠 Brain Tumor Detection

## 📌 Overview

This project uses Computer Vision and Deep Learning to detect brain tumors from MRI images.

It demonstrates image preprocessing and classification techniques used in healthcare AI systems.

---

## 🎯 Objective

- Detect brain tumors from MRI images
- Learn medical image classification
- Understand healthcare AI applications

---

## 🧠 Workflow

```text
MRI Image
↓
Image Preprocessing
↓
Deep Learning Model
↓
Tumor Detection
```
